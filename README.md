3Radio
======

F.A.Q
-----


**Q:** Why 3?

**A:** Because it *must* have only 3 hardware input devices.


Devices
-------

1. [RPi rev.2](http://www.raspberrypi.org/)
2. [LCD display](https://www.adafruit.com/products/499)
3. 3 [rotary encoders](http://www.adafruit.com/products/377)
4. Speakers.
5. Box.
